BuyEase is a full-stack e-commerce web application built using **ASP.NET Core MVC (.NET 9)**.
It demonstrates role-based authentication, structured MVC architecture, and complete product management functionality.


# Project Overview

This project includes:

* Role-based authentication (Admin & Customer)
* Secure authorization using `[Authorize(Roles = "Admin")]`
* Full CRUD operations for Products and Categories
* Shopping cart with session persistence
* Responsive UI built with Bootstrap
* Entity Framework Core (Code-First) with SQL Server


# Admin Features

* Dashboard with system overview
* Product and Category management (Create, Read, Update, Delete)
* Customer activity monitoring
* Protected admin routes with role-based access control


# Customer Features

* Product browsing and search
* Add to cart functionality
* User registration and login
* Account dashboard


# Tech Stack

* ASP.NET Core MVC (.NET 9)
* Entity Framework Core
* Microsoft SQL Server
* Bootstrap 5
* HTML, CSS, JavaScript
